# FAM 文件格式

## 格式观察

当前 demo 中 `.fam` 基本可按 plain text 处理。

线路 `Dev/*.fam` 样例显示，FAM 常见格式不是简单 `key=value`，而是三段式：

```text
标签=KEY=值
```

示例字段：

```text
电压等级=VOLTAGE=AC500kV
型号=TYPE=5TDZ-62、63-1
导线分裂数=BUNDLENUMBER=4
挂接点信息=WIREPOINT=...
```

需要注意：

- 标签可用于人工理解（界面展示）。
- KEY 适合进入字段字典。
- value 可能为空。
- 变电工程样本的key为中文，value用`/`代表空，变电工程的`FAM文件`可能内容完全为空。
- 线路工程样本的key为英文，第二个`=`无内容代表空。

当前 FAM 解析建议：

| 元素  | 说明                  |
| ----- | --------------------- |
| label | 第一个 `=` 前的标签   |
| key   | 第二个字段中的KEY     |
| value | 第二个 `=` 后的原始值 |

---

## 文件概述

FAM（Family/Attribute）文件是 GIM 工程中的属性定义文件，采用键值对文本格式。FAM 文件通过 `BASEFAMILY` 字段实现属性继承链，子级 FAM 文件可以引用父级 FAM 文件获取通用属性，同时定义自身特有的属性。

FAM 文件根据所处目录不同，其内容和组织方式有所差异：

| 所在目录     | 说明                                                       |
| ------------ | ---------------------------------------------------------- |
| **CBM 目录** | 结构较简单，主要定义 `BASEFAMILY` 继承关系，其他属性可为空 |
| **DEV 目录** | 包含详细的设备参数，按分类节组织                           |

## 文件格式

- **编码**：UTF-8
- **行分隔符**：换行符
- **键值分隔符**：`=`
- **节标题**：INI 风格的 `[节名]`（仅 DEV 目录下的 FAM 文件使用）

## 字段说明

### 通用字段

| 字段         | 格式         | 说明                                               |
| ------------ | ------------ | -------------------------------------------------- |
| `BASEFAMILY` | `<uuid>.fam` | 引用上级属性文件，形成属性继承链。可为空表示无上级 |

### DEV 目录下 FAM 文件的分类节

DEV 目录下的 FAM 文件使用 INI 风格的节标题将参数按工程阶段分类：

| 节标题       | 说明                                 |
| ------------ | ------------------------------------ |
| `[设计参数]` | 设计阶段参数，如额定电压、额定电流等 |
| `[产品参数]` | 产品制造参数，如型号、厂家等         |
| `[施工参数]` | 施工安装参数                         |
| `[测试参数]` | 调试测试参数                         |
| `[运检参数]` | 运行检修参数                         |

每个节下为标准的 `KEY=VALUE` 格式参数，键名和值均由具体设备类型决定。

## 引用关系

```
CBM 文件
└── BASEFAMILY → <uuid>.fam（CBM 目录下的 FAM）
                    └── BASEFAMILY → <uuid>.fam（上级 FAM，继承链）
                                        └── ...

DEV 文件
└── BASEFAMILY → <uuid>.fam（DEV 目录下的 FAM）
                    ├── BASEFAMILY → <uuid>.fam（上级 FAM）
                    ├── [设计参数]
                    ├── [产品参数]
                    ├── [施工参数]
                    ├── [测试参数]
                    └── [运检参数]
```

## 示例

### CBM 目录下的 FAM 文件

```
BASEFAMILY=a1b2c3d4-e5f6-7890-abcd-ef1234567890.fam
```

### DEV 目录下的 FAM 文件

```
BASEFAMILY=b2c3d4e5-f6a7-8901-bcde-f12345678901.fam

[设计参数]
额定电压=10kV
额定电流=630A
额定频率=50Hz
额定短路开断电流=25kA

[产品参数]
型号=ZN28A-12
厂家=某电气集团
出厂日期=2024-01
出厂编号=CB-2024-001

[施工参数]
安装方式=户内
安装高度=1.2m

[测试参数]
绝缘电阻=≥1000MΩ
工频耐压=42kV/1min

[运检参数]
检修周期=12个月
巡检周期=1个月
```
