# 已知限制

> 线路 GIM 可视化 MVP 阶段的已知限制。
> M4 路线图中的改进将逐步解决这些问题。

## 地图渲染

| 限制 | 说明 | M4 计划 |
|------|------|---------|
| 无真实底图 | Canvas 2D 绘制经纬度网格 + 边框，无卫星图/地形图/道路网 | M4-A: MapLibre + 离线瓦片 |
| 简化投影 | 等距投影（Equirectangular），高纬度地区有畸变 | M4-A: MapLibre 内置投影 |
| 无坐标偏移修正 | 直接使用 BLHA 原始坐标，未做 GCJ-02/WGS-84 转换 | M4-A: 坐标偏移处理 |

## 塔位

| 限制 | 说明 | M4 计划 |
|------|------|---------|
| 非真实模型 | 使用圆形/菱形符号，非真实塔型 3D 模型 | M4-B: 塔型符号细化 |
| 无 MOD 解析 | 未解析 .mod 几何文件，无法展示塔位详细几何 | M4-C: MOD 解析 |
| 塔型分类有限 | 仅区分直线塔（圆形）和耐张塔（菱形），未细分具体塔型 | M4-B: 基于 DEVICETYPE 细分 |

## 导线

| 限制 | 说明 | M4 计划 |
|------|------|---------|
| 折线渲染 | 两塔之间用直线连接，非真实弧垂 | M4-B: 悬链线计算 |
| 无悬链线 | 未使用 KVALUE/SPLIT 参数计算导线弧垂曲线 | M4-B: 悬链线 |
| 无样式区分 | 仅按 WIRETYPE 着色，未按电压等级/回路数区分 | M4-B: 样式区分 |
| 无断面图 | 未生成线路纵断面图 | M4-B: 断面图（可选） |

## 跨越点（CROSS）

| 限制 | 说明 | M4 计划 |
|------|------|---------|
| 部分无坐标 | 部分 CROSS 节点无 BLHA 坐标，无法在地图定位 | M4-C: CROSS 坐标补全 |
| 无跨越物类型 | 未区分跨越对象（电力线/通信线/道路/河流等） | M4-C: 属性增强 |

## IFC（变电工程）

| 限制 | 说明 | M4 计划 |
|------|------|---------|
| 无 Fragments 缓存 | 每次重新解析 IFC，加载较慢（ENABLE_FRAGMENTS_CACHE=false） | M4-D: 缓存工具 |
| 模型可能不完整 | web-ifc 解析部分 IFC 可能丢失构件（OBC 已知限制） | — |
| Fragments 异常 | "Malformed tile" 被 catch 不阻断，但该 IFC 可能显示不全 | M4-D: 日志等级 |

## 底图引擎

| 限制 | 说明 | M4 计划 |
|------|------|---------|
| 未引入地图引擎 | MVP 不使用 MapLibre/Leaflet/Cesium | M4-A: MapLibre 验证 |
| 无在线瓦片 | 离线运行，无法加载 OSM/卫星瓦片 | M4-A: 离线瓦片方案 |
| 无真实底图 overlay | 当前 Canvas 独立绘制线路图，尚未叠加 MapLibre/离线瓦片底图 | M4-A: Canvas overlay 方案 |

## 缓存

| 限制 | 说明 | M4 计划 |
|------|------|---------|
| 版本变化失效 | PARSER_VERSION 变更时所有旧缓存自动失效 | M4-D: 缓存迁移 |
| 无缓存管理工具 | 无法手动清除/导出缓存（仅 Ctrl+Shift+D 诊断） | M4-D: 缓存管理 UI |
| 首次导入较慢 | 大型线路工程 FAM/DEV 属性解析 + 入库可能耗时数秒 | M4-D: 性能优化 |

## 其他

| 限制 | 说明 | M4 计划 |
|------|------|---------|
| 单工程模式 | 同时只能打开一个 GIM 工程 | — |
| 无搜索功能 | 不支持按塔位编号/设备名称搜索定位 | M4-D: 搜索 |
| 无导出功能 | 不支持导出地图截图/属性表格 | M4-D: 导出 |
